Ripple
## Welcome to Ripple


### Island
##### *The part of the interface you interact with is called the Island.*


## There are 3 main modes in the interface


### Large Mode
##### *When you click on the Island this mode with be active. It's the mode you will use the most, and it contains all Tabs you use.*

### Quick Mode
##### *When you hover over the Island you'll see this. It contains the time, weather, and if you have battery alerts there, those will show too.*

### Still Mode
##### *It's mode you with see when you aren't doing anything with the Island.*


### These are the parts of the Island*


### Tabs
##### *The tabs are tiles that you switch through in the interface using the arrow keys.*

### Quick Apps
##### *A Tab with 4 apps of your choosing, so you can quickly access them.*

### Settings
##### It's the last tab and it allows you to adjust things from the Island's background color to you Quick Apps.

### Shortcuts
##### Just press Control + the number of the tab to.
